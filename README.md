# Spring Thymeleaf Template

Projeto Java 8 com Spring MVC e Thymeleaf

### Versão
1.0.1

## Clonando o projeto

Requer o Git configurado. No terminal, use o comando: 
git clone https://github.com/tiagosousaeti/spring-thymeleaf-template.git

## Dependências
- Java 8
- Servlet API 3.1.0
- Spring MVC 5.0.1
